Coursework

PROG 109: Intro To Web Development introduces fundamental programming and mark-up techniques for the Web. Students design procedures and write computer instructions to solve business related problems with a focus on simple applications and games. Students learn procedural programming, develop user interfaces, and work with events and objects. The class will teach client-side web development fundamentals.

This 5 credit course met Online only, Instructed by Adam Brehm

PROG 110: Intro To Programming uses a current object-oriented programming language. Students learn and apply the fundamental principles of programming. Students design solutions, write computer instructions to solve business problems, learn procedural programming, document and debug computer applications.

This 5 credit course met Online only, Instructed by Phil Duncan
